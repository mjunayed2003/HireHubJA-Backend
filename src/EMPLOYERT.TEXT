আপনার UI এর সাথে API এর কানেকশন গাইড:
১. Dashboard (উপরের ৪টি বক্স):
কল করুন: GET /employer/dashboard
রেসপন্স: { activeJobs: 4, pendingApplicants: 8, interviewsScheduled: 8, hiresCompleted: 4 }
এটি দিয়ে প্রথম স্ক্রিনশটের উপরের বক্সগুলো পূরণ করবেন।
২. My Job Post (কার্ডগুলো):
কল করুন: GET /employer/jobs
রেসপন্স: জবের লিস্ট এবং applicants: 06 সংখ্যাটি পাবেন। "View Applicants" বাটনে ক্লিক করলে জবের ID টি নিবেন।
৩. Post a Job (স্টেপ ফর্ম):
ফর্মের সব ডাটা কালেক্ট করে POST /employer/jobs-এ পাঠাবেন।
UI এর "Skills" এবং "Benefits" গুলোকে সুন্দর করে সাজিয়ে description এর মধ্যে HTML হিসেবে পাঠিয়ে দিতে পারেন অথবা আলাদা ফিল্ড চাইলে স্কিমা আপডেট করতে হবে। বর্তমান স্কিমা অনুযায়ী description এই সব দিতে হবে।
৪. Candidate Profile & Resume:
যখন "Details" বাটনে ক্লিক করবেন, তখন GET /employer/application/:id কল করবেন।
এখানে ইউজারের education, experience, skills এবং resumeUrl পাবেন যা দিয়ে সুন্দর প্রোফাইল পেজ (আপনার স্ক্রিনশটের মতো) বানাবেন।
৫. Schedule Interview Modal:
তারিখ, সময় এবং জুম লিংক নিয়ে POST /employer/application/:id/interview এ হিট করবেন।
এটি ডাটাবেসে Interview টেবিল আপডেট করবে এবং অ্যাপ্লিকেশন স্ট্যাটাস পরিবর্তন করবে।
৬. Interview Scheduled Page (সবার ইন্টারভিউ লিস্ট):
GET /employer/interviews কল করলে সব শিডিউল করা ইন্টারভিউয়ের লিস্ট আসবে যা আপনার "Interview Scheduled" পেজের কার্ডগুলোতে দেখাবেন।
এই কোডগুলো আপনার প্রজেক্টে src/employer ফোল্ডারে পেস্ট করুন এবং app.module.ts এ EmployerModule ইমপোর্ট করে নিন।
